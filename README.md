# Cinema TEST <br><br> [Demo](https://bexazak.github.io/cinema/)

## Stack
- Vue3, Composition API
- Vite
- Pinia
- Vue Router 4

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
